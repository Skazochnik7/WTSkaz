Игра "Тренер Ведьмы (русская редакция)"
Версия 1.3.1

Формат сохранений версии 1.3.1 частично не совпадает с форматом сохранений версии 1.3. Однако в новой версии можно безопасно использовать старые сохранения, если сохранение было сделано в определенной точке. Порядок действий такой: 
1. В версию 1.3 игры перейдите в ночное время суток. 
2. Завершите все текущие ивенты, если они есть: разговоры с Гермионой, Снейпом, прочитайте письма, заберите посылки. 
3. На экране игры должен отображаться только джинн, сидящий за столом. 
4. Сохранитесь. 
5. Это сохранение можно безопасно использовать в новой версии. 

Сохранение в любой другой точке при определенных условиях также может заработать в новой версии, если у вас нет других сохранений, попробуйте использовать то, что есть. Если при загрузке сохранения не появилась критическая ошибка, есть хороший шанс, что и дальше все будет работать корректно.

Свежую информацию о новых версиях игры всегда можно найти на нашем сайте http://wtrus.ixbb.ru/
